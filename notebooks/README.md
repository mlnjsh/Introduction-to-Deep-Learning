# Standalone Lecture Notebooks

Individual teaching notebooks used across the course sessions. These are self-contained
demonstrations (many with saved outputs). For the structured, fully-executed series see
[`../notebooks_DeepLearning/`](../notebooks_DeepLearning/) and
[`../notebooks_RNN_LSTM/`](../notebooks_RNN_LSTM/).

| Notebook | Topic |
|----------|-------|
| `0_Batch-MiniBatch-SGD.ipynb` | Batch vs mini-batch vs stochastic gradient descent |
| `1_Backpropogation_Coded.ipynb` | Backpropagation implemented from scratch |
| `SGD with Momentum from scratch.ipynb` | SGD + momentum, from scratch |
| `Steps to build Model.ipynb` | The end-to-end model-building workflow |
| `2_Solving_Loan_Prediction_problem_using_Neural_Network.ipynb` | Loan-approval MLP walkthrough |
| `Optimizers_MNIST.ipynb` | Comparing optimizers on MNIST |
| `Weight_Initialization_Techniques_in_Keras.ipynb` | Weight initialization in Keras |
| `Batch_Normalization_in_Neural_Networks_using_Keras.ipynb` | Batch normalization |
| `Dropout_in_Neural_Networks_using_Keras.ipynb` | Dropout regularization |
| `Early_stopping_in_Neural_Networks_using_Keras.ipynb` | Early stopping |
| `Gradient_Clipping_in_Keras.ipynb` | Gradient clipping |
| `21_Hyperparameters_Neural_Networks_using_Keras.ipynb` | Hyperparameter tuning |
| `Grid_Search.ipynb` | Grid search over hyperparameters |
| `filters.ipynb` | Visualizing CNN filters (`filterrs.py` helper) |
| `DL_Session1_Faculty_Notebook.ipynb` | Session 1 faculty notebook |

Supporting assets: `XOR_gate.png`, `dropout_comparision.png`, `filterrs.py`.

Datasets referenced by some notebooks (`loan_prediction_data.csv`, `housing.data.txt`) live at
the repository root.
